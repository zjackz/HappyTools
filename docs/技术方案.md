## 数据库配置

- CLI 通过 `backend/powerkit3000.consoleapp/appsettings.json` 中的 `ConnectionStrings:AppDb` 读取数据库连接信息；运行时可使用 `ConnectionStrings__AppDb` 环境变量覆盖。
- 修改连接字符串后须重新构建 CLI（`dotnet build powerkit3000.sln`），以确保最新配置被复制到输出目录。
- 最新数据库迁移 `MakeLocationOptional` 允许项目缺失地理信息；更新数据库时运行 `cd backend/powerkit3000.data && dotnet ef database update`。
